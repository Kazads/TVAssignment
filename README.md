# TvAssignment for AstraZenica
## Installation
* Install nodeJS, instructions can be found [here](https://nodejs.org/en/download)
* Run `npm install` in this folder
## Start application
* Run `npm start`
* When the application runs, open Google Crome at `localhost:3000`
## Run unit tests
* Run `npm test`
